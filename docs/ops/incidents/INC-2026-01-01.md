# Incident INC-2026-01-01

- Mức độ: SEV-3
- Thời gian: 2026-01-03 10:05-10:33
- Ảnh hưởng: latency endpoint search tăng mạnh (p99 ~ 1.4s trong 28 phút)

## Timeline
- 10:05: Alert `APILatencyP99High` kích hoạt.
- 10:10: On-call xác nhận spike query DB.
- 10:18: Thực hiện tạm thời tăng replica read DB.
- 10:25: Latency giảm dần.
- 10:33: Alert tự đóng.

## Root Cause
- Query search không dùng index tối ưu sau một thay đổi filter mới.

## Corrective Action
- Bổ sung composite index và cập nhật query planner hint.

## Preventive Action
- Thêm gate kiểm tra query plan vào pipeline trước release.
- Bổ sung cảnh báo sớm khi p95 vượt ngưỡng warning > 10 phút.
